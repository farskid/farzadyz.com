Statecharts are simplle, just kidding. they can be intimidating at first.
Here is an example:

```js viz test
import { createMachine } from "xstate";

const statechart = createMachine({
  initial: "a",
  id: "test",
  states: {
    a: {
      on: {
        GO: "b",
      },
      after: {
        2000: "b",
      },
    },
    b: {
      on: {
        GO: "a",
      },
    },
    c: {},
    d: {},
  },
});
```

You can visualize them!
